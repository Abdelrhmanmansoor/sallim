## 🎮 الميزة الجديدة 1: لعبة العيدية (Eidiya Game)

### 📝 الفكرة
لعبة تفاعلية تتيح لزوار الديوانية الإجابة على أسئلة وكسب عيدياً افتراضية مع رسوم متحركة وتأثيرات صوتية ممتعة.

### 🔧 التطبيقات التقنية

#### Backend (server/routes/diwaniya.js)
```javascript
// نقاط API الجديدة:
1. GET /api/v1/diwaniya/:username/game
   - الحصول على أسئلة اللعبة
   
2. GET /api/v1/diwaniya/:username/game/status
   - التحقق من إمكانية اللعب
   
3. POST /api/v1/diwaniya/:username/game/answer
   - إرسال إجابة مع حماية من السبام
   
4. PUT /api/v1/diwaniya/:username/game
   - تحديث أسئلة اللعبة (للمالك)
   
5. GET /api/v1/diwaniya/:username/game/stats
   - الحصول على إحصائيات (للمالك)
```

#### Frontend Components
```javascript
// مكون جديد:
src/components/EidiyaGame.jsx
- مرحلة التعهد (Pledge)
- مرحلة اللعب (Playing)
- مرحلة النتيجة (Result)
- رسوم متحركة للمال (Canvas)
- تأثيرات صوتية
- أزرار المشاركة والتحميل
```

#### API Utilities (src/utils/api.js)
```javascript
// دوال جديدة:
getEidiyaGame(username)
getEidiyaGameStatus(username, sessionId)
submitEidiyaGameAnswer(username, data)
updateEidiyaGame(username, data)
getEidiyaGameStats(username)
```

### 🔒 الحماية الأمنية

```javascript
// 1. Rate Limiting
- حد: 10 إجابات في الدقيقة
- أداة: express-rate-limit

// 2. Session Tracking
- معرف جلسة فريد لكل لاعب
- منع اللعب المتكرر

// 3. Anti-Cheat
- منع الإجابة على نفس السؤال أكثر من مرة
- التحقق من الإجابة على السيرفر

// 4. One-Time Play
- كل لاعب يمكنه اللعب مرة واحدة فقط
- تخزين في localStorage أو session

// 5. Input Validation
- استخدام Joi للتحقق من جميع المدخلات
- سلسلة التحقق من جانب السيرفر
```
