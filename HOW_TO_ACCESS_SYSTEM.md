## 🔐 طريقة الدخول (المُحدثة!) ✨

### 🎯 طريقة واحدة للدخول لجميع الحسابات

**الخطوة 1:** افتح المتصفح واذهب إلى:
```
http://localhost:5173/login
```

**الخطوة 2:** اختر نوع الحساب من الأزرار:
- **مستخدم** - للمستخدمين العاديين
- **شركة** - للشركات المسجلة

**الخطوة 3:** أدخل بيانات الدخول:
- البريد الإلكتروني
- كلمة المرور

**الخطوة 4:** اضغط "تسجيل الدخول"

**سيتم التوجيه تلقائياً إلى:**
- **مستخدم** → `/dashboard/diwaniya`
- **شركة** → `/company/dashboard`

**ملاحظة:** للدخول كـ Admin، استخدم رابط مباشر: http://localhost:5173/admin/dashboard

## 🆕 الميزات الجديدة في White Label

### للـ Admin (سَلِّم):

1. **إدارة الثيمات** 🎨
   - إنشاء ثيمات جديدة
   - تعديل وحذف الثيمات
   - تحديد حالة الثيم (عام/حصري/مخفي)
   - تعيين ثيمات حصرية لشركات معينة

2. **إدارة الباقات** 📦
   - إنشاء باقات جديدة
   - تحديد عدد البطاقات والأسعار
   - تحديد الميزات والحدود
   - تطبيق باقة على شركة

3. **إدارة الشركات المحسّنة** 🏢
   - تتبع رصيد البطاقات
   - تحديث اشتراك الشركات
   - إضافة رصيد للشركات
   - تقارير تفصيلية لكل شركة

### للشركات:

1. **إرسال بالجملة** 📤
   - رفع ملف CSV/Excel
   - إرسال بطاقات لعدد كبير
   - تتبع حالة الإرسال

2. **تقارير الإرسال** 📊
   - جدول بجميع البطاقات المرسلة
   - معرفة من فتح البطاقة
   - تصفية وتصدير CSV

## ⚡ بدء سريع

### إذا تريد تجربة النظام الآن:

1. **افتح المتصفح:** http://localhost:5173

2. **اضغط F12** لفتح Developer Tools

3. **اذهب إلى Console** والصق:
```javascript
// عرض جميع الثيمات
fetch('http://localhost:3001/api/v1/admin/themes')
  .then(r => r.json())
  .then(d => {
    console.log('📊 الثيمات المتاحة:', d.data.themes)
    alert(`وجدنا ${d.data.total} ثيم!`)
  })

// عرض جميع الباقات  
fetch('http://localhost:3001/api/v1/admin/packages')
  .then(r => r.json())
  .then(d => {
    console.log('📦 الباقات المتاحة:', d.data.packages)
    alert(`وجدنا ${d.data.total} باقة!`)
  })

// عرض جميع الشركات
fetch('http://localhost:3001/api/v1/admin/companies')
  .then(r => r.json())
  .then(d => {
    console.log('🏢 الشركات المسجلة:', d.data.companies)
    alert(`وجدنا ${d.data.total} شركة!`)
  })
```
